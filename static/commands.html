How to Use Commands

Send these commands directly to the bot in your Telegram chat with @Elizabeth_Olsen_robot.

/start

Start the bot and get a welcome message

/start
/help

Get a list of available commands and usage information

/help
/setcaption

Set a default caption for all media in a channel

/setcaption This is my awesome caption with {file_name} placeholder
/delcaption

Delete the current caption for a channel

/delcaption
/all

Process all media in a channel with the current caption

/all
/save_template

Save a caption template for later use

/save_template my_template This is a template for {file_name}
/list_templates

List all saved caption templates

/list_templates
/view_template

View a specific caption template

/view_template my_template
/use_template

Apply a saved template to a channel

/use_template my_template
